## 要件
- **h2見出しのみ**で構成する（h3は使わない）
- h2は5〜8個（まとめ含む）
- 見出しは短くキャッチーに（20文字以内推奨）
- 見出しに絵文字を1つ入れる（💰✨🔥📱🎯💡など）
- 読者の疑問や感情に寄り添う見出しにする（「○○って実際どう？」「ぶっちゃけ○○は…」など）
- 上位記事の内容をカバーしつつ、会話調で差別化する
- 最後のh2は「✨ まとめ」にする
- 「①②③」のような番号付けは不要

### 良い見出し例
- 💰 気になる料金は？
- 🔥 使ってみて驚いたこと
- 🤔 デメリットも正直に言います
- 📱 始め方はめちゃ簡単
- ✨ まとめ

### 悪い見出し例（WordPress SEO記事っぽい → NG）
- Antigravityの料金体系とGoogle Oneでお得に使う仕組み3つ
- 料金以上の価値を感じる圧倒的なメリット5つ

## 出力フォーマット (JSON)
以下のJSON形式で出力してください。JSON以外のテキストは不要です。
{
  "outline": [
    {
      "h2": "見出しテキスト",
      "h3s": [],
      "diagramDescription": ""
    }
  ]
}

※ h3sは必ず空配列、diagramDescriptionは必ず空文字にしてください。
